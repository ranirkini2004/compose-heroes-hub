# 🦸 Compose Heroes Hub

A modern Superheroes Directory app built using **Jetpack Compose**.

## ✨ Features

- 🔍 Search superheroes
- ❤️ Mark as favourite
- 📄 Detail screen
- ♿ Accessibility support
- 🧭 Navigation component
- 🧠 ViewModel state management
- 🎨 Material 3 design

## 🏗 Architecture

- MVVM
- Repository pattern
- Navigation Compose
- Coil for image loading

## 📸 Screens

- Heroes List Screen
- Hero Detail Screen

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose
- Coil
- ViewModel
